简介：此项目是一个在线代码测评系统（oj），我负责设计web端的包括前端和后端，也就是说这些代码全部由我独立编写。涉及用户管理（登录注册以及用户会话session）模块，问题模块（问题展示编辑代码区和提交代码），用户正确比率排行模块，提交状态模块，以及讨论板等待开发模块。

缺点：虽然前端设计的有些简单，功能模块也还不多。未做到较好的封装性以代码的利用率。


可能遇到的问题：用于使用的是php较老的版本，在php7.0及以上版本，有些函数有了更改，如果移植到高版本的php环境下，需要调试。如对于MySQL的连接就有mysql_connect和mysqli_connect的区别和函数参数也有区别。

想加还没加的功能：讨论板、列表的翻页功能，用户权限管理，用户对自己以往提交过得代码修改，对自己编译错误对应提交的反馈内容展示，关于我们和使用教程的页面展示

使用教程：
1，设有游客模式，有浏览网页的权限
2，注册账号，有编辑代码并提交检测的权限，以后还会设置管理员权限，管理员权限有添加题目、管理其他用户权限更改一些页面样式的权限，现未设管理员权限，任何用户拥有所有权限
3，进入问题，选择题目，提交，编辑代码，提交，跳转到状态页面观看对应提交的反馈。
4，排行榜可浏览所有用户的实时答题情况
5，讨论板等正在开发中。
